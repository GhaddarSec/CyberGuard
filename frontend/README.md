CyberGuard:
CyberGuard is a web application designed to promote cyber awareness. It provides practical guidance on protecting personal information, understanding cyber threats, and improving digital hygiene for individuals and organizations. The app includes key features like Security Tips, Quizzes, and Updates & Alerts. The website is built using React, Tailwind CSS, and Vite, with a clean, responsive design.

Setup Instructions:
Clone the repository, install dependencies, and run the development server:

1-git clone https://github.com/GhaddarSec/CyberGuard.git
2-cd CyberGuard
3-npm install
4-npm run dev
